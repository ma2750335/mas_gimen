# 奇門遁甲：現代金融量化交易規則 (v2.6)

本文件集結了「奇門符號金融解碼」與「精準進退場 SOP」，是進行黃金 (XAU/USD) 與白銀 (XAG/USD) 量化分析的最高導引。

## 第一章：奇門符號的現代金融翻譯 (情報解碼)

### 1. 九宮與八卦 (價格框架 / 支撐阻力)
九宮八卦代表市場的「價格框架」，即技術圖表的關鍵價位。
- **坎一宮 (屬水)**：低谷、盤整打底。
- **坤二宮 (屬土)**：緩慢孕育，長線買盤進場 (做多有利)。
- **震三宮 (屬木)**：強勁爆發、價格快速拉升。
- **巽四宮 (屬木)**：趨勢成形、市場情緒跟風。
- **中五宮 (土)**：寄於坤宮，重大趨勢轉折點。
- **乾六宮 (屬金)**：高點天花板、主力所在位 (價格尾數帶 6)。
- **兌七宮 (屬金)**：破壞與震盪、急跌或洗盤。
- **艮八宮 (屬土)**：停止趨勢、強大阻力位 (Resistance)。
- **離九宮 (屬火)**：最狂熱高潮、泡沫即將破滅。

### 2. 十天干：三奇六儀 (參與者與資金流)
- **甲**：大趨勢絕對方向。
- **乙 (日奇)**：溫和上漲、利多消息。
- **丙 (月奇)**：報復性買盤、強力動能。
- **丁 (星奇)**：精準轉折點、技術利多數據。
- **戊 (資本)**：代表 **本金與保證金 (Margin)**。
- **己 (陷阱)**：暗藏風險、誘多誘空、洗盤圈套。
- **庚 (敵人/賣壓)**：市場大魔王。
- **辛 (錯誤)**：判斷失誤或短線價格修正。
- **壬 (流動性)**：行情流暢順行、趨勢如水。
- **癸 (困境)**：行情停滯、交易卡住 (如清晨)。

### 3. 天盤庚 vs 地盤庚
- **天盤庚 (動態殺傷力)**：主動現價賣壓、機構倒貨、急跌，適合順勢做空，禁止接單。
- **地盤庚 (靜態鐵板)**：歷史阻力/支撐（如 $2000、$2500 等關卡），易引發劇烈反彈或震盪。

### 4. 八門、九星、八神 (行為、環境、心理)
- **八門 (交易行為)**：生門/開門 (買氣湧入)、死門 (買盤枯竭/做空)、景門 (虛假繁榮/警示)。
- **九星 (宏觀/央行政策)**：天心 (央行政策)、天蓬 (波動率風險指標)、天衝 (急漲跌爆發力)、天芮 (衰退指標)。
- **八神 (心理手法)**：值符 (絕對主趨勢)、螣蛇/白虎 (洗盤掃損/兇險急跌)、玄武 (誘騙假突破)、九天 (強烈衝刺)。

## 第二部分：每日實戰解盤 SOP (時盤推演四招)

### 第 1 招：找「用神」（確認主角是誰）
- **日干 (交易者我)**：確認我的狀態與氣場環境。
- **時干 (行情趨勢)**：標的物當下的具體動能。
- **戊 (資金利潤)**：看錢在哪裡。

### 第 2 招：看「五行生剋」（環境天時判斷）
- **時干生日干 (大吉)**：代表行情來生助你，老天送錢，勝率極高，獲利輕鬆。
- **你剋時干 (中吉)**：代表你可以掌控行情，可以賺到錢但過程波折，需精細盯盤。
- **時干剋你 (大凶)**：市場在針對你，行情跟你作對，這代表「市場不賞飯吃」，強烈建議空手觀望。

### 第 3 招：審視時干宮之「神、星、門」（判定多空方向）
- **判定要做多還是做空**：九天/值符、生門/開門屬多頭；九地/白虎、死門屬空頭。玄武螣蛇則代表洗盤。

### 第 4 招：檢查有沒有「庚」（排雷系統）
- **天盤庚**：減輕部位，預防砸盤。
- **地盤庚**：不宜追高，留意歷史鐵板。

## 第三部分：精準進退場時間 (應期) 與價格點位

1. **進場時間 (Energy Timing)**：
    - 利用十二地支。當時間地支與有利宮位五行共鳴時出手。

2. **退場時間 (Profit Taking)**：
    - 當時間走到剋制我方利多宮位之地支時，應平倉了結。

3. **價格點位 (Price Levels)**：
    - 利用九宮數字。參考尾數帶有該宮位數字（如乾六宮 -> 2016, 2060）的價格帶最為阻力或停利參考。
